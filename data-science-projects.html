Senior Thesis: Machine Learning Analysis of LiDAR Wetlands Data

Building a tool to analyze National Ecological Observatory Network full-waveform LiDAR data in a novel data format called Pulsewaves, which currently does not exist. K-means clustering and regression techniques will be used to associate the LiDAR data with biomass, and then biomass with carbon flux data to ultimately build a map of wetland carbon sequestration.

Convolutional Neural Network and Visual Transformer for Dog Breeds

Predicted the classes of 7 different dogs based on 9000+ dog training images by building a CNN and Transformer architecture on Pytorch achieving a 0.85 AUROC score.

Kernel Methods and Logistic Regularized Regression for Hospital Mortality
